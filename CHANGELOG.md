# Changelog

All notable changes to the **Oracle APEX LLM Chatbot Plugin** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-12

### Added
- Initial release of the Oracle APEX LLM Chatbot Plugin.
- Region Type Plugin with RAG/LLM capabilities.
- Support for Arabic (RTL) and English (LTR) via Plugin Attributes.
- Modern CSS styling with interactive loading animations and glassmorphic elements.
- Client-side Markdown rendering fallback and cleaning capabilities.
- APEX process integration out-of-the-box (`AI_CHAT_PROCESS`).
- Open-source GitHub repository with complete DevRel documentation structure.
